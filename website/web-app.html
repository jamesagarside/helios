Helios in Your Browser

A full ground control station that runs directly in your web browser. No installation required.

How the Web App Works

Step 1

Start the relay

Run a lightweight relay server on the computer connected to your drone. The relay bridges your MAVLink connection to a WebSocket that the browser can reach.

Step 2

Open the web app

Visit app.heliosgcs.com in any modern browser. Enter the relay address and connect. Works on desktop, tablet, and mobile.

Step 3

Fly

Monitor live telemetry, view your aircraft on the map, plan missions, and control flight modes - all from your browser with no software to install.

What You Can Do in the Browser

Live Telemetry

Real-time attitude, altitude, speed, GPS position, battery, and flight mode display - the same instruments as the desktop app.

Live Map

Track your aircraft position on an interactive map with heading indicator, breadcrumb trail, and mission waypoint overlay.

Mission Planning

Create waypoint missions, upload them to your flight controller, and monitor mission progress - all from the browser.

Flight Mode Control

Switch flight modes, arm and disarm, trigger return-to-launch, and access emergency controls directly from the browser interface.

Any Device

Works on any device with a modern browser. Use your laptop, tablet, or even a phone as a secondary monitoring display.

No Installation

Nothing to download, nothing to update. Just open the URL and connect. Perfect for field use when you need a quick ground control station.

When to use the web app vs the desktop app

The web app is ideal for quick monitoring, sharing a view with a team member, or using a device where you can't install software. For the full Helios experience - including DuckDB flight recording, USB serial connections, offline maps, and video streaming - use the desktop application.
